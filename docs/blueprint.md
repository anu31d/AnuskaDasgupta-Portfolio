# **App Name**: Anuska's Digital Portfolio

## Core Features:

- About Section: Displays a concise bio pulled from the provided LinkedIn profile.
- Experience Showcase: Highlights professional journey and achievements, showing experience, education, certifications, and projects.
- Skills and Achievements: Displays technical proficiencies and accomplishments through a dedicated skills section.
- Visual Gallery: Showcases personal interests, style, and activities through a gallery section with pictures.
- AI-Powered Content Generation: Summarizes key accomplishments and skills to create personalized portfolio section descriptions. This acts as a tool to provide content to display on each tab.

## Style Guidelines:

- Primary color: Vibrant violet (#7F5AF0) to convey innovation and modernity.
- Background color: Dark charcoal (#1E1E2F) to ensure excellent contrast.
- Accent color: Pale pink (#F1C1F1) to give a soft, feminine touch to the futuristic design.
- Headings font: 'Space Grotesk' (sans-serif) for a modern, tech-oriented feel.
- Body font: 'Inter' (sans-serif) to ensure readability and a clean aesthetic. 
- Note: currently only Google Fonts are supported.
- Use minimal vector icons in a neon style to complement the futuristic theme.
- Employ a dynamic layout with clear, tabbed sections for easy navigation: About, Experience, Education, Certifications, Projects, Skills, and Gallery.
- Add subtle animations on page load and tab transitions to enhance the user experience.
